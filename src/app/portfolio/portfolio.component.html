ProApp

O ProAPP é uma aplicação desenvolvida para fornecer autenticação segura e eficiente, utilizando JWT Tokens. Este projeto foi construído com .NET 8 e o framework ASP.NET Core, garantindo uma base sólida, escalabilidade e desempenho otimizado.

Framework Backend: Asp.Net Core

Utilizei o poderoso framework Asp.Net Core para criar uma base sólida para o aplicativo usando .Net 8, garantindo escalabilidade e desempenho otimizado.

Páginas Razor: Razor Pages

Desenvolvi interfaces de usuário dinâmicas e interativas com o uso das Razor Pages, garantindo uma experiência fluida e responsiva para os usuários.

Autenticação com Jwt Tokens: Jwt Tokens

Implementei um sistema de autenticação avançado e seguro, utilizando Jwt Tokens, garantindo a integridade e a confidencialidade dos dados dos usuários.

Notes API

A Notes API é uma aplicação desenvolvida para organizar anotações, oferecendo uma solução robusta e escalável para o gerenciamento eficiente de dados.

Este projeto foi construído utilizando tecnologias modernas e seguindo as melhores práticas de desenvolvimento de software.

.NET Core 8

Plataforma de desenvolvimento de alto desempenho e com suporte para construção de aplicações escaláveis.

Arquitetura Limpa

Separação clara de responsabilidades com camadas distintas (Core/Domain, Infrastructure, Application, API/Interface), promovendo alta testabilidade e manutenção do código.

CQRS (Command Query Responsibility Segregation)

Separação de operações de leitura e escrita, otimizando o desempenho e a organização do código.

Banco de Dados SQL Server

Utilização de banco de dados relacional confiável e eficiente.

Entity Framework Core

ORM robusto para mapeamento objeto-relacional, facilitando a interação com o banco de dados.

Padrão Repository

Abstração da lógica de acesso aos dados, promovendo um design desacoplado e fácil de testar.

Validação de Dados

Implementação de validações consistentes com o FluentValidation, garantindo a integridade dos dados.

Testes Unitários

Adoção de práticas de testes unitários utilizando XUnit e Moq para assegurar a qualidade e a confiabilidade do código.
